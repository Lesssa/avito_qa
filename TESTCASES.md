### TC-001: Создание объявления с корректными данными
- **Предусловия**: API доступно.
- **Шаги**:
  1. Отправить POST-запрос на "/api/1/item" с корректными данными:
        "sellerID": 123650,
        "name": "Aboba@134",
        "price": 50,
        "statistics": {
            "contacts": 5,
            "likes": 50,
            "viewCount": 100
        }
  3. Проверить статус ответа.
- **Ожидаемый результат**: Статус ответа 200.
